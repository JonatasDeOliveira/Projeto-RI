27.02.2012. Появилась возможность фильтровать рейтинг решений задачи по языку программированию
С сегодняшнего дня вы можете фильтровать рейтинг решений задачи по языку программирования. Это особенно актуально для вас, если вы решаете задачи на Java или C#, у которых минимальное время работы решения выше, чем, например, у C++. Если вы решите задачу на нескольких языках программирования, то лучшие ваши решения по каждому языку попадут в соответствующие фильтрованные рейтинги, а лучшее из всех решений попадет в общий рейтинг.

Чтобы открыть рейтинг решений задачи, перейдите по соответствующей ссылке внизу условия задачи.

21.11.2007. Изменены опции компилятора C/C++
В командную строку компилятора C/C++ была добавлена опция /O1. Эта опция используется вместо опции по умолчанию /O2. Описание из справки компилятора:
  • /O1 – оптимизировать для наилучшей скорости, но отключить некоторые оптимизации, которые увеличивают размер кода ради небольшого выигрыша в скорости
  • /O2 – оптимизировать для наилучшей скорости (по умолчанию)
С опцией /O2 компилятор ICL иногда генерирует неверный код. Для того чтобы предотвратить неправильные вердикты проверяющей системы мы изменили опции, несмотря на небольшую потерю в скорости работы некоторых решений. Текущая командная строка компилятора:
  • Для C:
    icl /GX /TC /Za /O1 /D ONLINE_JUDGE
  • Для C++:
    icl /GX /TP /Za /O1 /D ONLINE_JUDGE
