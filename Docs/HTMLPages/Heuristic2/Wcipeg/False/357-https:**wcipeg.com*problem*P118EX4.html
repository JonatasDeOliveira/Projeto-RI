p118ex4: Times Tables

Write a program that reads one positive integer N and then prints its times table containing values up to N x N.

Sample Input

5

Sample Output

5 X 1 = 5
5 X 2 = 10
5 X 3 = 15
5 X 4 = 20
5 X 5 = 25
