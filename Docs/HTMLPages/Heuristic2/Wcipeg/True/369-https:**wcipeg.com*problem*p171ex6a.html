p171ex6a - Max

Given T real numbers, find and output the largest one to four decimal places.

Input

The first line contains a single integer T (1 ≤ T ≤ 1000000). Each of the following T lines contains one real number xi (-106 ≤ xi ≤ 106).

Output

A single line: the largest of the T real numbers.

Sample Input

3
1.5
3
2.2

Sample Output

3.0000
