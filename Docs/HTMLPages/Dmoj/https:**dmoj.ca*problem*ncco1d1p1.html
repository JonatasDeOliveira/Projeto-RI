Mock CCO '18 Contest 1 Problem 1 - A Geometry Problem

View as PDF
Submit solution
All submissions
Best submissions

Points: 20 (partial)
Time limit: 1.0s
Memory limit: 16M

Problem type
Geometry

Given an axis-aligned rectangle with vertices at (0,0) and (L,W) and N closed disks of radius 100, determine if there exists a path starting at (0,ys) and ending at (L,ye) where 0≤ys,ye≤W and where the path does not visit a point strictly outside the given rectangle or touching any of the disks.

In the event no such path exists, compute the minimum number of disks to remove such that a path can be constructed without exiting the rectangle or touching the remaining disks.

Constraints

1≤W≤50⋅103

1≤L≤50⋅103

1≤N≤250

0≤Xi≤L

0≤Yi≤W

Input Specification

The first line will contain three space-separated integers, L, W, and N.

Each of the next N lines contains a pair of integers Xi and Yi, indicating a center of one of the N disks.

Output Specification

Print, on a single line, the minimum number of disks to remove such that there exists a path connecting the left side of the rectangle to the right side of the rectangle that does not exit the rectangle or touch any of the disks.

Sample Input

Copy
130 340 5
10 50
130 130
70 170
0 180
60 260

Sample Output

Copy
1
